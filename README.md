# Git-Workshop

# Cloning a repo
```
git clone <repo uri>
```
# Make some changes
```
console.log("Hello <Your name>!")
```
# Stage your changes
```
git add .
```
# Commit your changes
```
git commit -m "Added Hello World"
```
Commit is like making a save file, youll always be able to go back and look at your old commits
# Push your commit
```
git push
```
